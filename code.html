my gists

Whenever I need to figure how to perform a programming task, I write small, self-contained, examples and post the result on github gist. Includes C/C++ interface for HDF5, boost::MPI, and python snippets for multiple libraries.

cascading_detection

This repository contains a python implementation of the cascading detection algorithm. It is a thin wrapper around other existing community detection algorithms. It keeps track of nodes' identity through multiple pass of these algorithms, and produces extensive logs containing: time spent on each pass, edge list and clusters detected at each pass, etc. The wrapper currently handles CFinder (Palla et al.), link clustering (Ahn et al.) and GCE (Lee at al.), i.e. all the algorithms that are analyzed in this article.

Structural Preferential Attachment (SPA+)

SPA+ is stochastic growth process that produces realistic networks with a modular structure. A minimal version was initially introduced by Laurent Hébert-Dufresne et al. in a series of papers back in 2011-2012. While this initial version, dubbed SPA, reproduced most mesoscopic properties of real networks accurately, it made strong assumptions about the structure of the communities of a network: namely that real communities are formed by fully connected cliques of nodes (a hypothesis reminiscent of the work of Palla et al. [2005], for example). This often not the case. This observation prompted the development of an extended model (SPA+) that accounts for the heterogeneous nature of community density in real networks. See my publications page for more information.

The hyper-linked repository contains a robust C++ implementation of both the simple model (SPA) and the extended model (SPA+).

Hierarchical Preferential Attachment

The Hierarchical Preferential Attachment model (HPA) is a direct extension of SPA, and contains the latter as a special case. It is described in length in our Phys. Rev. E. paper. The hyper-linked repository does not contain a full implementation of the model (yet?), but provides useful analytical tools to play around with the mean-field equations.
